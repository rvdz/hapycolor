class hapycolor.filters.reducer.Reducer¶

Bases: hapycolor.filters.base.Filter

static apply(palette)¶

Returns a reduced list of rgb colors. The output result contains the maximal number of colors of the provided list, where each color is at at least a distance threshold from the others. This distance is currently a constant integer which defines the minimal CIEL2000 distance between colors.

Parameters:palette – the input palette
Returns:the output palette
Return type:an instance of Palette
